# left behind / still here

photography by brook rodia. cleveland heights, ohio.

## development

```
pnpm install
pnpm dev
```

## adding images

drop images into `public/images/{series-name}/` and update the `src` fields in `src/data/galleries.ts`.

series directories:
- `work` — master sequence
- `god-is-indifferent` — the neighborhood at midnight
- `the-world-continued` — abandoned spaces
- `dear-fellow-member` — st. marks presbyterian church
- `what-the-light-chose` — close detail work
